# slim-controller-connector

Slim Controller Connector is an extension to the Slim Framework to provide a simple way
to connect controller with a Slim route.

## Install

With Composer.

	"require": {
		"slim/slim": "~2.3",
		"rgsone/slim-controller-connector": "dev-master"
	}
